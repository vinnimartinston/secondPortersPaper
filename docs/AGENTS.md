# Agent instructions (Paper2)

When you **add, change, or refactor methods** in this repository:

1. Read [`guidelines.md`](guidelines.md) in this directory (`docs/`) first and follow whatever is documented there.
2. For rationale, background, or longer explanations, read [`guidelines-with-explanations.md`](guidelines-with-explanations.md) here (especially if something in the short guidelines is unclear).

Treat those files as the source of truth for how methods should be shaped once they are filled in.
